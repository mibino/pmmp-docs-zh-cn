PocketMine-MP 文档
===========================

本仓库的source文件夹为 http://doc.pmmp.io/ 的源文件.

文档服务来自 https://readthedocs.org/.

Build local
-----------

### Pre-requisites
- python3
- pip
- install the requirements using `pip install -r requirements.txt`

完成以上操作后，可以通过运行“makehtml”来构建站点。

HTML输出文件将放置在“/build`目录。
